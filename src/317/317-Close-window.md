# Close Window
This packet is sent when a player presses the close, exit or cancel button on an interface.

## Packet Details
| Key | Value |
|--|--|
| Name | Close window |
| Description | Sent when a player presses the close, exit or cancel button on an interface. |
| Opcode | 130 |
| Type | Fixed |
| Length | 0 |
| Revision | 317 |
