# Interface offset
Sets the offset for drawing of an interface.

## Packet Details
| Key | Value |
|--|--|
| Name | Interface offset |
| Description | Sets the offset for drawing of an interface |
| Opcode | 70 |
| Type | Fixed |
| Length | 6 |
| Revision | 317 |
