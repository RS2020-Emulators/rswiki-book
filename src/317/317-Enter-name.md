# Enter name
Sending the packet to the client will make the client open up the "Enter name" interface for things such as friend-adding.

## Packet Details
| Key | Value |
|--|--|
| Name | Enter name |
| Description | Opens up the name entry interface. |
| Opcode | 187 |
| Type | Fixed |
| Length | 0 |
| Revision | 317 |
