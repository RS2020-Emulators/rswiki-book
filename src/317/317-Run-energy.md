# Run Energy
Sends how much run energy the player currently has.

## Packet Details
| Key | Value |
|--|--|
| Name | Run energy |
| Description | Sends the players run energy level. |
| Opcode | 110 |
| Type | Fixed |
| Length | 1 |
| Revision | 317 |

## Packet Structure
| Data Type | Description |
|--|--|
| [Byte](/Data-Types.html#common-data-types) | The energy level. |
